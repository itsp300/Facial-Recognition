# Facial-Recognition
## Modules to Install
### Install opencv
```
pip install opencv-contrib-python --upgrade 
or without extra modules:  
pip install opencv-python 
```

### Install cmake
```
pip install cmake
```

### Install dlib
```
pip install dlib
```

### Install numpy
```
pip install numpy
```

### Install face_recognition
```
pip install face_recognition
```
